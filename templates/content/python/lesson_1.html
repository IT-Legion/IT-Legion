Привет мир!

Существует традиция которая гласит о том что изучение любого языка программирования начинается с написания программы 'Привет мир'

Python хорош тем что для написания такой программы нам потребуется всего одна встроенная функция

print("Привет мир!")

Вот и вся магия

4 типа данных

  • string: Строчное значение из себя представляет набор символов

  • integer: Целочисленное число представляет из себя число

  • floot: Числа с плавующей точкой

  • bool: Тип данных выражает своё значение в виде правды или ложь
