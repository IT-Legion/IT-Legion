abstract

Ключевое слово abstract используется для объявления абстрактного класса или метода. Абстрактный метод не имеет реализации и должен быть переопределен в подклассах.

and

Ключевое слово and используется для логического объединения двух выражений. Оно возвращает true, если оба выражения истинны, и false в противном случае.

array

Ключевое слово array используется для создания массива в PHP. Массивы могут содержать несколько значений под одним именем переменной.

as

Ключевое слово as используется для присвоения значения элементу массива при итерации в цикле foreach.

break

Ключевое слово break используется для выхода из ближайшего цикла for, foreach, while или do-while.

callable

Ключевое слово callable используется для объявления типа аргумента или возвращаемого значения, который должен быть вызываемым (функцией или методом).

case

Ключевое слово case используется в конструкции switch для определения блока кода, который будет выполнен, если значение соответствует выражению в case.

catch

Ключевое слово catch используется для перехвата исключений, которые были выброшены в блоке try.

class

Ключевое слово class используется для объявления нового класса в PHP. Классы позволяют создавать новые типы объектов, которые могут содержать данные и методы для работы с этими данными.

clone

Ключевое слово clone используется для создания копии объекта. Клонированный объект является новым объектом с таким же содержимым.

const

Ключевое слово const используется для определения константы класса. Константы класса не могут быть изменены после их объявления.

continue

Ключевое слово continue используется для пропуска текущей итерации цикла и перехода к следующей итерации.

declare

Ключевое слово declare используется для установки директив выполнения для блока кода, таких как строгие типы или включение тиков.

default

Ключевое слово default используется в конструкции switch для определения блока кода, который будет выполнен, если ни одно из значений case не соответствует.

do

Ключевое слово do используется для создания цикла do-while, который выполняется по крайней мере один раз и продолжается до тех пор, пока условие истинно.

else

Ключевое слово else используется в конструкции if для выполнения блока кода, если условие ложно.

elseif

Ключевое слово elseif используется для проверки дополнительного условия в конструкции if.

enddeclare

Ключевое слово enddeclare используется для завершения блока declare.

endfor

Ключевое слово endfor используется для завершения блока for.

endforeach

Ключевое слово endforeach используется для завершения блока foreach.

endif

Ключевое слово endif используется для завершения блока if.

endswitch

Ключевое слово endswitch используется для завершения блока switch.

endwhile

Ключевое слово endwhile используется для завершения блока while.

extends

Ключевое слово extends используется для создания класса, который наследует методы и свойства другого класса.

final

Ключевое слово final используется для предотвращения переопределения метода или наследования класса.

finally

Ключевое слово finally используется для выполнения блока кода после выполнения try и catch, независимо от того, было ли выброшено исключение.

fn

Ключевое слово fn используется для создания стрелочных функций (анонимных функций) в PHP.

for

Ключевое слово for используется для создания цикла, который повторяется определенное количество раз.

foreach

Ключевое слово foreach используется для итерации по элементам массива или объекта.

function

Ключевое слово function используется для определения функции в PHP.

global

Ключевое слово global используется для доступа к глобальным переменным внутри функции или метода.

goto

Ключевое слово goto используется для перехода к метке в коде. Использование goto может ухудшить читаемость кода и его следует избегать.

if

Ключевое слово if используется для проверки условия. Если условие истинно, выполняется блок кода, связанный с if.

implements

Ключевое слово implements используется для реализации интерфейса в классе.

include

Ключевое слово include используется для включения и выполнения кода из другого файла.

include_once

Ключевое слово include_once используется для включения и выполнения кода из другого файла, если он не был включен ранее.

instanceof

Ключевое слово instanceof используется для проверки, является ли объект экземпляром определенного класса или его подкласса.

insteadof

Ключевое слово insteadof используется для разрешения конфликтов при использовании трейтов, указывая, какой метод следует использовать вместо другого.

interface

Ключевое слово interface используется для определения интерфейса, который может быть реализован классом.

isset

Ключевое слово isset используется для проверки, установлена ли переменная и не равна ли она null.

list

Ключевое слово list используется для присвоения переменных из массива в одну операцию.

match

Ключевое слово match используется для создания выражений соответствия, которые похожи на switch, но возвращают значение.

namespace

Ключевое слово namespace используется для объявления пространства имен, чтобы избежать конфликтов имен функций, классов и констант.

new

Ключевое слово new используется для создания нового экземпляра класса.

or

Ключевое слово or используется для логического объединения двух выражений. Оно возвращает true, если хотя бы одно из выражений истинно, и false в противном случае.

print

Ключевое слово print используется для вывода строки на экран. Оно всегда возвращает 1.

private

Ключевое слово private используется для объявления свойств или методов, которые могут быть доступны только внутри класса, в котором они объявлены.

protected

Ключевое слово protected используется для объявления свойств или методов, которые могут быть доступны внутри класса, в котором они объявлены, и в его подклассах.

public

Ключевое слово public используется для объявления свойств или методов, которые могут быть доступны из любого места.

require

Ключевое слово require используется для включения и выполнения кода из другого файла. Если файл не найден, выполнение скрипта завершится с фатальной ошибкой.

require_once

Ключевое слово require_once используется для включения и выполнения кода из другого файла, если он не был включен ранее. Если файл не найден, выполнение скрипта завершится с фатальной ошибкой.

return

Ключевое слово return используется для завершения выполнения функции и возврата значения функции.

static

Ключевое слово static используется для объявления статических свойств и методов класса, а также для сохранения значения переменной в локальной области видимости между вызовами функции.

switch

Ключевое слово switch используется для выполнения одного из множества блоков кода в зависимости от значения выражения.

throw

Ключевое слово throw используется для выброса исключения. Исключения используются для обработки ошибок и других необычных ситуаций.

trait

Ключевое слово trait используется для объявления трейтв, которые позволяют включать методы в классы с целью повторного использования кода.

try

Ключевое слово try используется для начала блока кода, в котором могут быть выброшены исключения, которые затем перехватываются блоками catch.

unset

Ключевое слово unset используется для удаления переменной или элемента массива.

use

Ключевое слово use используется для включения трейтов в класс, а также для импорта пространств имен и их сокращений.

var

Ключевое слово var является устаревшим и используется для объявления свойств класса. Вместо него рекомендуется использовать public, protected или private.

while

Ключевое слово while используется для создания цикла, который выполняется до тех пор, пока условие истинно.

xor

Ключевое слово xor используется для логического исключающего ИЛИ (исключающее ИЛИ) двух выражений. Оно возвращает true, если одно из выражений истинно, но не оба.

yield

Ключевое слово yield используется для генерации значений в генераторе, который позволяет итерировать через последовательность значений.

yield from

Ключевое слово yield from используется для делегирования части генератора другому генератору или итерируемому объекту.
