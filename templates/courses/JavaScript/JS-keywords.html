break

Выход из цикла или оператора switch.

case

Определяет блок кода в операторе switch.

catch

Определяет блок кода для обработки исключений.

class

Определяет класс.

const

Объявляет константу.

continue

Переход к следующей итерации цикла.

debugger

Останавливает выполнение и вызывает функцию отладки.

default

Определяет блок кода по умолчанию в операторе switch.

delete

Удаляет объект, свойство объекта или элемент массива.

do

Определяет цикл do-while.

else

Определяет блок кода, который будет выполнен, если условие в if ложно.

export

Экспортирует модуль или часть модуля.

extends

Используется для создания дочернего класса на основе родительского.

finally

Определяет блок кода, который будет выполнен после блока try и catch.

for

Определяет цикл с предопределенными условиями.

function

Объявляет функцию.

if

Определяет условный оператор.

import

Импортирует модули или части модуля.

in

Проверяет наличие свойства в объекте.

instanceof

Проверяет, является ли объект экземпляром класса.

let

Объявляет локальную переменную с блочной областью видимости.

new

Создает новый экземпляр объекта.

return

Возвращает значение из функции.

super

Вызывает конструктор родительского класса.

switch

Определяет оператор switch для многократного выбора.

this

Ссылается на текущий объект.

throw

Выбрасывает исключение.

try

Определяет блок кода для обработки исключений.

typeof

Возвращает тип переменной или выражения.

var

Объявляет переменную.

void

Выражение, которое вычисляет значение undefined.

while

Определяет цикл while с предусловием.

with

Устанавливает область видимости для выражений.

yield

Приостанавливает и возобновляет выполнение генератора.
