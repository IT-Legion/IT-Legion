True и False

Ключевые слова True и False используются для обозначения булевых значений в Python. Они представляют два значения логического типа: истину и ложь соответственно.

None

Ключевое слово None используется для обозначения отсутствия значения или null-значения. Оно часто применяется в качестве значения по умолчанию для аргументов функций или для инициализации переменных, которым будет присвоено значение позже.

and

Ключевое слово and используется для логического объединения двух выражений. Оно возвращает True, если оба выражения истинны, и False в противном случае.

as

Ключевое слово as используется для присваивания псевдонима в таких конструкциях, как импортирование модулей с новым именем или использование менеджеров контекста с блоком with.

assert

Ключевое слово assert используется для отладки программ. Оно позволяет проверять условие, и если условие не выполняется, возникает исключение AssertionError.

async

Ключевое слово async используется для объявления асинхронной функции, которая позволяет выполнять неблокирующие операции с помощью await.

await

Ключевое слово await используется внутри асинхронной функции для приостановки выполнения до завершения awaitable объекта, такого как асинхронная функция или корутина.

break

Ключевое слово break используется для выхода из ближайшего цикла for или while раньше, чем условие цикла станет ложным.

continue

Ключевое слово continue используется для пропуска текущей итерации цикла и перехода к следующей итерации.

class

Ключевое слово class используется для объявления нового класса в Python. Классы позволяют создавать новые типы объектов, которые могут содержать данные и методы для работы с этими данными.

def

Ключевое слово def используется для определения функции в Python. Оно задает блок кода, который может быть вызван по имени с использованием круглых скобок.

del

Ключевое слово del используется для удаления объектов в Python. Оно может удалять переменные, элементы списков или словарей и т. д.

elif

Ключевое слово elif (сокращение от "else if") используется для проверки дополнительного условия в конструкции if.

else

Ключевое слово else используется в конструкции if для выполнения блока кода, если все предшествующие условия не были истинными.

except

Ключевое слово except используется для обработки исключений в блоке try. Оно указывает, какой код должен быть выполнен, если возникает исключение.

finally

Ключевое слово finally используется для определения блока кода, который будет выполнен независимо от того, возникло исключение в блоке try или нет.

for

Ключевое слово for используется для создания циклов, которые повторяются для каждого элемента в последовательности, такой как список или строка.

from

Ключевое слово from используется с import для импорта конкретных частей модуля.

global

Ключевое слово global используется для объявления глобальных переменных внутри функции или другого блока кода.

if

Ключевое слово if используется для проверки условия. Если условие истинно, выполняется блок кода, связанный с if.

import

Ключевое слово import используется для импорта модулей в текущий скрипт.

in

Ключевое слово in используется для проверки наличия элемента в последовательности, такой как список или строка.

is

Ключевое слово is используется для проверки идентичности объектов, т.е. находятся ли они в одной и той же области памяти.

lambda

Ключевое слово lambda используется для создания небольших анонимных функций.

nonlocal

Ключевое слово nonlocal используется для объявления переменных, которые находятся в объемлющем (но не глобальном) пространстве имен.

not

Ключевое слово not используется для логического отрицания. Оно возвращает True, если выражение ложно, и False в противном случае.

or

Ключевое слово or используется для логического объединения двух выражений. Оно возвращает True, если хотя бы одно из выражений истинно.

pass

Ключевое слово pass используется как заглушка для создания пустых блоков кода, которые не выполняют никакие действия.

raise

Ключевое слово raise используется для возбуждения исключений.

return

Ключевое слово return используется для возврата значения из функции.

try

Ключевое слово try используется для оборачивания блока кода, в котором могут возникнуть исключения. Оно должно сопровождаться блоком except, finally, или else.

while

Ключевое слово while используется для создания циклов, которые продолжаются до тех пор, пока условие истинно.

with

Ключевое слово with используется для оборачивания блоков кода, которые требуют начальной и завершающей обработки, таких как работа с файлами.

yield

Ключевое слово yield используется для возврата значения из генераторной функции и приостановки её выполнения, сохраняя её состояние для последующего продолжения.
