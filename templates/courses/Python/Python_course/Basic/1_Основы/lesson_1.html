О языке: Python

Python — это высокоуровневый, интерпретируемый язык программирования общего назначения, с динамической строгой типизацией и автоматическим управлением памятью. Язык является полностью объектно-ориентированным в том плане, что всё является объектами.

История

Python был разработан в конце 1980-х годов, Гвидо ван Россумом в CWI - Национальный исследовательский институт математики и информатики в Нидерландах.
Первая версия (0.9.0) была выпущена в феврале 1991 года. За годы своего существования Python прошел через несколько версий:
Python 1.x (1991-2000): первая версия языка с базовыми возможностями.
Python 2.x (2000-2010): значительное обновление, включающее многие новые возможности и улучшения. Поддержка Python 2 была официально прекращена в январе 2020 года.
Python 3.x (с 2008 года): текущая версия языка, которая включает многие улучшения и изменения, несовместимые с Python 2.x. Python 3.x продолжает развиваться и является рекомендуемой версией для новых проектов.
