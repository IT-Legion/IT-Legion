Традиция: Привет мир!

Традиционно изучение любого языка программирования начинается с приветствия.
Давайте посмотрим, как это делается на Python.

Для начала убедимся в том, что питон установлен.
Открайте терминал и введите следующую команду: 

python3 --version

Если Python не установлен или команда не распознана, вы получите примерно такое сообщение: 
command not found: python

В этом случае, нужно зайти на официальный сайт и скачать дестрибудив в соответсвии с вашей операционной системой.
Как правило установка дестрибудива не является сложной задачей темнемение лучше чтоб она была краткая: видео инструкция.

Зеверщающим этапом этого урока осталось попрактиковатся в написании программ.
Для этого в терминале активируте интерпретатор языка программирования Python.
Введите в терменале: 

python3
Консоль переходит в интерактивный режим (REPL - Read-Eval-Print Loop), где можно вводить Python-код, который сразу будет исполняться. Вы увидите приглашение ввода, обычно это >>>.

Познакомимся с первой функцией.

Введите: 
>>> print("Hello, World!")
В консоль вернётся:
#Hello, World!

Функция print() в Python используется для вывода данных на стандартный вывод, который обычно представляет собой экран консоли или терминала.

Синтаксис

print(*objects, sep=' ', end='\n', file=sys.stdout, flush=False)

Аргументы

  • *objects: Один или несколько объектов, которые нужно вывести. Объекты могут быть различного типа: строки, числа, списки и т.д. Все они будут преобразованы в строки с помощью функции str и выведены.
  • sep (по умолчанию ' '): Строка-разделитель, которая вставляется между объектами. По умолчанию это пробел.
  • end (по умолчанию '\n'): Строка, которая добавляется в конец вывода. По умолчанию это символ новой строки.
  • file (по умолчанию sys.stdout): Объект файлового типа, в который будет производиться вывод. По умолчанию это стандартный вывод (консоль).
  • flush (по умолчанию False): Булевый флаг, определяющий, нужно ли принудительно сбросить (очистить) буфер вывода. Полезно для вывода данных в реальном времени.
